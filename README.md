# react-native-draggable-fab

Draggable FAB for react native

## Installation

```sh
npm install react-native-draggable-fab
```

## Usage

```js
import DraggableFab from "react-native-draggable-fab";

// ...

const result = await DraggableFab.multiply(3, 7);
```
